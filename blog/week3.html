This week I learned about css positioning. The positions relative, fixed, and absolute allow us to control the layout of out page. There are other ways to do page layout such as flexbox but when we use these positions we can use the four attributes: left, right, top, and bottom which give us power to place elements of the pages where we want.                 document.querySelectorAll('nav a')
                    .forEach(e => e.addEventListener('click', _ => change(e.dataset.id)))
            
            
                function change(n) {
                    let text = document.querySelectorAll('main div')
                    text.forEach(p => p.classList.remove('active'))
                    text[n - 1].classList.add('active')
                }
